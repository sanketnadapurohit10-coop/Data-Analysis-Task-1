# Data Analysis Task 1: Data Cleaning and Preprocessing

## Objective
The objective of this task is to clean and preprocess the Customer Personality Analysis dataset using Microsoft Excel. The dataset was prepared by handling missing values, checking for duplicate records, standardizing column names, converting date formats, and verifying data types.

## Dataset
- **Original Dataset (Kaggle):** https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis
- **Cleaned Dataset (GitHub):**  https://github.com/sanketnadapurohit10-coop/Data-Analysis-Task-1/blob/main/customer_personality_analysis_cleaned.xlsx

## Data Cleaning Steps
- Checked and handled missing values.
- Filled missing values in the **Income** column.
- Checked for duplicate records (no duplicates found).
- Standardized column headers by converting them to lowercase with underscores.
- Converted the **Dt_Customer** column to a proper date format.
- Verified data types for all columns.
- Reviewed categorical columns for consistency.
